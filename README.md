Knight Might Fight is a web-based game that is played in a browser, so no installation is required. Instead, users can simply follow this link: https://waterded.itch.io/sdev265-game. Once the page loads, the player will click “Start,” and select a difficulty level. When the game starts, the user can pick a weapon and hit the “roll” button for their turn. After their turn, the enemy will attack, and this loop will continue until the player wins or dies.
